"Game: The Game" is a platforming game I made to test myself, I haven't done anything like this before and I had a lot of fun working on it
-------------------------------------------------------------------------------------------------------------------------------------------

Navigate through 5 short levels, each with their own gimmick. Jump on enemies, collect coins, get extra jumps (if the level has them) and find secrets.
--------------------------------------------------------------------------------------------------------------------------------------------------------

Controls\
Left and Right = Movement\
Space = Jump\
Down = Crouch
----------------------------
